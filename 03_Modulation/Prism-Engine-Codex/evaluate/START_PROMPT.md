# Evaluate Start Prompt

进入 `evaluate/` 目录后，可对 Codex 输入：

```text
Read ../shared/prompts/evaluate.md and audit the target files. Produce a structured report in ../reports with PASS, CONDITIONAL, or FAIL, plus concrete revision steps.
```

如果审计长篇章节，可补充：

```text
Use ../novels/<project>/outline.md and ../novels/<project>/story_bible.md as continuity references.
```
