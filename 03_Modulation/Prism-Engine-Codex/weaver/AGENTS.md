# Weaver 引擎作用域

- 先阅读 `../shared/prompts/weaver.md`、`../specs/schema_outline.md`、`../specs/schema_story_bible.md`。
- 长篇正文以 Scene Shards 形式写入 `../novels/{project}/chapters/Chapter_XX/Scene_YYY.md`。
- 每次写完场景后保持可检查状态；章节产物通过 `../scripts/compile_chapter.sh` 生成。
- 续写前读取 `outline.md`、`story_bible.md` 与上一场景或上一章。
- 未经用户要求，不在本目录直接承担总控调度职责。
